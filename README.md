🏦 Mini Bank Management System (Python - OOP)
📌 Project Overview
   This is a console-based banking system developed using Python and Object-Oriented Programming principles. The system allows             dministrators to manage customer accounts efficiently with persistent data storage.

🚀 Features
 ~ Create new bank account
 ~  Display all accounts
 ~ Search account by account number
 ~ Deposit money
 ~ Withdraw money (with minimum balance validation)
 ~ Delete account
 ~ Transaction history with date & time tracking
 ~ Data persistence using pickle

🛠 Technologies Used
~ Python
~ Object-Oriented Programming (OOP)
~ File Handling (pickle)
~ Datetime module

💡 Key Concepts Applied
~ Class design and encapsulation
~ Data persistence
~ List-based transaction tracking
~ Exception handling
~ Modular programming

~ This project strengthened my understanding of:
~ Real-world system modeling
~ State management in objects
~ File-based data storage
~ Structured program design
